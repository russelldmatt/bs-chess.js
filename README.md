# [Chess.js](https://github.com/jhlywa/chess.js) bindings for Bucklescript in Reason

Work in progress

# Build
```
yarn install
yarn build
```

# Build + Watch
```
yarn run start
```

# Run (Node)
```
node src/Chess.bs.js
```

# Run (Browser)
Works now (thanks @chenglou!).
```
yarn run webpack
open index.html
```
